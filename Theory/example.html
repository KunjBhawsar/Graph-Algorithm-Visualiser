Solved Examples

1. BFS Example

Problem:

Perform BFS traversal on the following graph starting from vertex 1:

Graph Edges: 1→2, 1→3, 2→4, 3→4, 3→5, 4→5

Solution:

Step 1: Start at vertex 1. Queue: [1], Visited: {1}
Step 2: Dequeue 1, explore neighbors 2 and 3. Queue: [2, 3], Visited: {1, 2, 3}
Step 3: Dequeue 2, explore neighbor 4. Queue: [3, 4], Visited: {1, 2, 3, 4}
Step 4: Dequeue 3, explore neighbor 5 (4 already visited). Queue: [4, 5], Visited: {1, 2, 3, 4, 5}
Step 5: Dequeue 4, no new neighbors. Queue: [5], Visited: {1, 2, 3, 4, 5}
Step 6: Dequeue 5, no neighbors. Queue: [], Visited: {1, 2, 3, 4, 5}
BFS Traversal Order: 1 → 2 → 3 → 4 → 5

2. DFS Example

Problem:

Perform DFS traversal on the same graph starting from vertex 1:

Graph Edges: 1→2, 1→3, 2→4, 3→4, 3→5, 4→5

Solution:

Step 1: Start at 1, mark visited. Visited: {1}
Step 2: Visit neighbor 2 from 1. Visited: {1, 2}
Step 3: Visit neighbor 4 from 2. Visited: {1, 2, 4}
Step 4: Visit neighbor 5 from 4. Visited: {1, 2, 4, 5}
Step 5: Backtrack to 4, then 2, then 1
Step 6: Visit neighbor 3 from 1. Visited: {1, 2, 4, 5, 3}
DFS Traversal Order: 1 → 2 → 4 → 5 → 3

3. Prim's Algorithm Example

Problem:

Find the Minimum Spanning Tree using Prim's algorithm starting from vertex A:

Weighted Edges:
A-B: 4, A-C: 3
B-C: 1, B-D: 2
C-D: 4, D-E: 2
C-E: 5

Solution:

Step Action MST Edges Total Cost
1 Start at A. Add edges A-B(4), A-C(3) to PQ - 0
2 Select A-C(3). Add C to MST A-C 3
3 Add edges from C: C-B(1), C-D(4), C-E(5) A-C 3
4 Select C-B(1). Add B to MST A-C, C-B 4
5 Add edge B-D(2) A-C, C-B 4
6 Select B-D(2). Add D to MST A-C, C-B, B-D 6
7 Add edge D-E(2) A-C, C-B, B-D 6
8 Select D-E(2). Add E to MST A-C, C-B, B-D, D-E 8
MST Edges: A-C(3), C-B(1), B-D(2), D-E(2) | Total Cost: 8

4. Kruskal's Algorithm Example

Problem:

Find the Minimum Spanning Tree using Kruskal's algorithm for the same graph:

Weighted Edges:
A-B: 4, A-C: 3
B-C: 1, B-D: 2
C-D: 4, D-E: 2
C-E: 5

Solution:

Step Edge Weight Action Sets
1 B-C 1 Accept (no cycle) {A}, {B,C}, {D}, {E}
2 B-D 2 Accept (no cycle) {A}, {B,C,D}, {E}
3 D-E 2 Accept (no cycle) {A}, {B,C,D,E}
4 A-C 3 Accept (no cycle) {A,B,C,D,E}
5 A-B 4 Reject (creates cycle) {A,B,C,D,E}
6 C-D 4 Reject (creates cycle) {A,B,C,D,E}
7 C-E 5 Reject (creates cycle) {A,B,C,D,E}
MST Edges: B-C(1), B-D(2), D-E(2), A-C(3) | Total Cost: 8

Note: Both Prim's and Kruskal's algorithms produce the same MST cost (8), but may select different edges in different order while achieving the same optimal result.
